# 제주코딩베이스캠프 섹션 1 문제 21

## 1. 문제 설명

#### 문제 21. 다음 중 set을 만드는 방법으로 올바른 것을 모두 고르시오.        

        1)  var x = {1, 2, 3, 5, 6, 7};
          
        2)  var x = {};
          
        3)  var x = new Set('javascript');
          
        4)  var x = new Set(range(5));
          
        5)  var x = new Set();

## 2. 내 풀이

#### 문제 21.
        es6에 새로 등장한 set의 존재를 몰라서 1), 2) 으로 찍었다. 

## 3. 강의 해설

#### 문제 21.

        3)  var x = new Set('javascript');
    
        5)  var x = new Set();

        정답은 '3번', '5번'이다.
