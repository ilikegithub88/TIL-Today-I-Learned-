# 제주코딩베이스캠프 섹션 1 문제 24

## 1. 문제 설명

#### 문제 24. 이름이 입력되면 전부 대문자로 출력되는 프로그램을 만들어주세요.

        입력 : mary

        출력 : MARY

## 2. 내 풀이

        let name=prompt('이름 입력하세요');

        let changeName=name.toUpperCase();

        console.log(changeName);

## 3. 강의 해설

        var name = prompt("이름을 입력하세요.");

        console.log(name.toUpperCase());
