# 제주코딩베이스캠프 섹션 1 문제 23

## 1. 문제 설명

#### 문제 23. (O X 문제) console.log(10/3)의 출력 결과는 3이다.
   
## 2. 내 풀이

        X

## 3. 강의 해설

        정답은 'X'입니다. 

        출력 결과는 3.3333333333333335 이 나옵니다.

        소숫점이 없는 정수를 출력하고자 할 때는 Math.floor() 함수를 쓰면 됩니다.
