# 제주코딩베이스캠프 섹션 1 문제 2

## 1. 문제 설명

    'pass' 부분에 배열 내장함수를 이용하여 코드를 입력하고 다음과 같이 출력되게 하세요

    <데이터>

    var arr=[200,100,300];

    //pass

    console.log(arr);

    <출력>

    [200,100,10000,300]

## 2. 내 풀이

    arr.splice(2,0,10000); 

## 3. 강의 해설

    arr.splice(2,0,10000);
