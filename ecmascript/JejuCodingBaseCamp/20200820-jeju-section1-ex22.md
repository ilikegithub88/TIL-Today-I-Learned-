# 제주코딩베이스캠프 섹션 1 문제 21

## 1. 문제 설명

#### 문제 21. 다음 중 변수 i가 6의 배수인지 확인하는 방법으로 올바른 것은?

        1)  i / 6 == 0
          
        2)  i % 6 == 0
         
        3)  i & 6 == 0
         
        4)  i | 6 == 0
          
        5)  i // 6 == 0
   
## 2. 내 풀이

        2)  i % 6 == 0

## 3. 강의 해설

        2)  i % 6 == 0

        정답은 '2번'입니다.
